Visual Studio Code

官网：https://code.visualstudio.com/

VSCode可以认为是一个迷你版的Visual Studio，大概45M左右，拥有海量的插件可以满足PHP、JAVA、C#、JavaScript、Typescript 、Go、C/C++等语言开发和调试，支持智能提示等。

软件安装

进入官网下载软件，建议安装完成之时将所有的复选框都选上，方便以后的开发和使用。
