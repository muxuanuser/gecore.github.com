ionic 集成极光推送插件

本文档主要介绍极光推送插件的安装使用，ionic环境搭建和ionic的项目创建不在本文范畴。

1. 注册和使用

  1. 打开[‘极光推送官网’](‘https://www.jiguang.cn/')，注册并登陆，创建一个app。

  1. 读取重要信息，包括appkey和你的应用包名，这两个缺一不可。

2. 创建一个ionic项目

创建一个ionic项目

ionic start jgPush

3. 修改项目

修改config.xml文件，修改widget节点下面的id属性。

4. 安装插件

安装极光推送插件(请使用管理员身份)。

ionic cordova plugin add jpush-phonegap-plugin --variable APP_KEY=your_jpush_appkey

npm install --save @jiguang-ionic/jpush

插件安装完毕就可以编写代码了。

5. 编译项目

编译调试项目。（安卓为例）

ionic cordova build android

6. 测试

在此使用的是在线调试，当然可以使用本团队的api项目调试，[api.lucoder.com/doc/jpush](‘api.lucoder.com/doc/jpush’)调试，具体规则打开网站查看。
